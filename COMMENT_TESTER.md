## 📋 5 ÉTAPES POUR TESTER

### ⏰ Étape 1: ATTENDRE 3 MINUTES
Le déploiement Vercel est en cours. Patientez...

### 🔍 Étape 2: PAGE DE DIAGNOSTIC
Ouvrez cette URL dans votre navigateur:
```
https://votre-app.vercel.app/check-version.html
```

**Vérifiez**:
- ✅ Toutes les vérifications sont vertes
- ✅ Version du script: 2.0.2 ou supérieure
- ❌ Si erreur rouge: Suivez les instructions affichées

### 🗑️ Étape 3: VIDER LE CACHE (SI NÉCESSAIRE)
**Si la page de diagnostic montre des erreurs**:

#### Windows/Linux:
```
1. Ctrl + Shift + Delete
2. Cocher "Images et fichiers en cache"
3. Période: "Toutes les périodes"
4. Cliquer "Effacer"
```

#### Mac:
```
1. Cmd + Shift + Delete
2. Cocher "Images et fichiers en cache"
3. Période: "Toutes les périodes"
4. Cliquer "Effacer"
```

### 🔄 Étape 4: FERMER ET ROUVRIR
1. Fermez **TOUS** les onglets de l'application
2. Fermez **complètement** le navigateur
3. Rouvrez-le

### 🧪 Étape 5: TESTER L'APPLICATION
1. Ouvrir: `https://votre-app.vercel.app/`
2. Appuyer sur `F12` (DevTools)
3. Onglet "Console"
4. **Vérifier: AUCUNE erreur rouge**
5. **Vérifier: Message "Application prête."**
6. Cliquer sur "Section des Garçons"
7. Sélectionner: Classe PEI3
8. Sélectionner: Section A
9. Sélectionner: Élève Bilal Molina
10. **Vérifier: 9 cartes de matières s'affichent**
11. Cliquer sur: 📐 Mathématiques
12. **Vérifier: Formulaire s'affiche**
