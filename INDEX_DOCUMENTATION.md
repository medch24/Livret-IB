## 🎯 Par Besoin

### "J'ai un problème avec..."

#### ➡️ Les cartes de matières ne deviennent pas vertes
- Lire : **CORRECTIONS_FINALES_V2.0.4.md** (Section 1)
- Vérifier : La version déployée est 2.0.4

#### ➡️ Les notes ne s'affichent pas dans le tableau arabe
- Lire : **CORRECTIONS_FINALES_V2.0.4.md** (Section 2)
- Vérifier : Version 2.0.4 déployée

#### ➡️ Les observations arabes ne sont pas en RTL dans Word
- Lire : **GUIDE_RTL_TEMPLATE_WORD.md** ⭐⭐⭐
- Action : Modifier le template Word

#### ➡️ L'ordre des matières n'est pas bon dans le livret
- Lire : **ORDRE_MATIERES_WORD.md**
- Vérifier : Les noms des matières dans la DB

#### ➡️ Les noms de matières ont changé dans la DB
- Lire : **TRI_FLEXIBLE_MATIERES.md** ⭐
- Bonne nouvelle : Le système gère automatiquement les variations !

#### ➡️ Des matières vides apparaissent dans le livret
- Lire : **CLARIFICATION_MATIERES_LIVRET.md** ⭐
- Note : Seules les matières remplies devraient apparaître
